\doxysubsection{Detailed Description}
Construct a piecewise clothoids $ G(s) $ composed by 3 clothoid and one line segment that solve the G2 problem

{\bfseries{match}}

\[
\begin{array}{ll}
\textrm{endpoints:}\quad&
\begin{cases}
  G(0) = \mathbf{p}_0 & \\[0.5em]
  G(L) = \mathbf{p}_1 &
\end{cases}
\\[1em]
\textrm{angles:}\quad&
\begin{cases}
  \theta(0) = \theta_0 & \\[0.5em]
  \theta(L) = \theta_1 &
\end{cases}
\\[1em]
\textrm{curvature:}\quad&
\begin{cases}
  \kappa(0) = \kappa_0 & \\[0.5em]
  \kappa(L)  = \kappa_1 &
\end{cases}
\end{array}
\]

{\bfseries{Reference}}

The solution algorithm is described in


\begin{DoxyItemize}
\item {\bfseries{E.\+Bertolazzi, M.\+Frego}}, On the G2 Hermite Interpolation Problem with clothoids Journal of Computational and Applied Mathematics, vol 341, pp. 99-\/116, 2018
\end{DoxyItemize}
